# SeminarioCulturaDigitale
Intervention Proposal for Body Dysmorphic Disorder (BDD)

Developed an original research proposal exploring the use of the Metaverse as a complementary therapeutic tool for adolescents with Body Dysmorphic Disorder (BDD).

The project combines Virtual Reality, avatar-based interactions, cognitive behavioral therapy (CBT), and body image rehabilitation to design an immersive treatment framework. The proposal includes the design of therapeutic virtual environments, patient interaction workflows, four immersive treatment modules, evaluation methodology, ethical considerations, limitations, and future research directions.

This academic project integrates research from psychology, psychiatry, Human–Computer 
Interaction (HCI), Virtual Reality, and UX Design to investigate innovative digital approaches for mental healthcare.
